## Unit 1 — Foundations

### Sets & Set Notation

| Symbol | Name | Meaning | Example |
|---|---|---|---|
| `{ }` | Roster Form | Lists elements in a set | `{1, 2, 3}` |
| `∈` | Element Of | Is an element of a set | `2 ∈ {1,2,3}` |
| `∉` | Not an Element Of | Is not an element of a set | `5 ∉ {1,2,3}` |
| `…` | Ellipsis | Pattern continues indefinitely | `{1,2,3,…}` |

### Union & Intersection

| Symbol | Name | Meaning | Example |
|---|---|---|---|
| `∪` | Union | OR — combine both sets (no duplicates) | `A ∪ B` |
| `∩` | Intersection | AND — elements in both sets | `A ∩ B` |
| `∅` | Empty Set | Set containing no elements | `A ∩ B = ∅` |

### Complement

| Symbol | Name | Meaning | Example |
|---|---|---|---|
| `U` | Universal Set | All possible elements in the problem | `U = {1,2,3,4}` |
| `X′` | Complement of X | Elements in U not in X | `X′` |

### Set-Builder Notation

| Symbol | Name | Meaning | Example |
|---|---|---|---|
| `{x \| condition}` | Set-Builder Form | Defines a set using a rule | `{x \| x ≤ 3}` |

Read as: *the set of all x such that the condition is true.*

## Unit 2 — Probability

| Symbol | Name | Meaning |
|---|---|---|
| `P(A)` | Probability of A | Chance event A occurs |
| `P(A ∪ B)` | Union Probability | A or B occurs |
| `P(A ∩ B)` | Intersection Probability | A and B both occur |
| `A′` | Complement of A | Event A does not occur |

## Unit 3 — Descriptive Statistics

### Parameter vs. Statistic

| Measure | Population (Parameter) | Sample (Statistic) |
|---|---|---|
| Mean | μ (mu) | x̄ (x-bar) |
| Standard Deviation | σ (sigma) | s |
| Variance | σ² | s² |
| Proportion | p | p̂ |
| Size | N | n |
| Correlation | ρ (rho) | r |
